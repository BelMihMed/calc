# Калькулятор экономии от внедрения BitrixGPT

Интерактивный калькулятор экономического эффекта от внедрения ИИ **BitrixGPT** в Битрикс24.
Самодостаточный фронтенд (HTML + CSS + JS), **без бэкенда** — все расчёты в браузере.

## Возможности
- Сценарии, сгруппированные: «Сценарии в CRM», «ИИ-агенты», «Другие сценарии», «VibeCode».
- Липкая панель навигации с быстрыми переходами и живым итогом «Эффект: X ₽/мес».
- Поля коэффициентов вводятся в процентах; мгновенный пересчёт, разделители разрядов, валидация.
- У каждого блока и у итога — иконка «i» с понятным пояснением, как считается экономия.
- Итог: экономия ФОТ, прирост выручки, общий эффект (месяц/год) и сравнение со стоимостью подписки.
- Экспорт результата в **PDF** с выбором тарифа Битрикс24 и названием компании.
- У ряда блоков «Подробнее» открывает отдельный лендинг со сценариями (в новом окне).

## Структура репозитория
```
index.html                      ← калькулятор (это и есть сайт GitHub Pages)
.nojekyll                       ← отключает Jekyll
golos-bitrix24.html             ← лендинг: Голосовое управление Битрикс24
podderzhka-bitrix24.html        ← лендинг: Агент первой линии поддержки
assistant-bitrix24.html         ← лендинг: Личный ассистент
zapis-bitrix24.html             ← лендинг: Агент онлайн-записи
documents-bitrix24.html         ← лендинг: Генерация документов
README.md
docs/
  TZ.md                         ← техзадание: формулы, сценарии, эталонные примеры
build/
  build_landings.py             ← генератор лендингов (ассистент/запись/документы)
  build_landing_support.py      ← генератор лендинга поддержки
```

> Важно: ссылки «Подробнее» в калькуляторе **относительные**, поэтому все 6 HTML-файлов
> (`index.html` + 5 лендингов) должны лежать **в корне репозитория**, рядом друг с другом.

## Публикация на GitHub Pages
1. Залейте содержимое этой папки в репозиторий (минимум — корневые `.html` и `.nojekyll`).
2. **Settings → Pages → Build and deployment**: Source = *Deploy from a branch*, Branch = **main**, папка = **/ (root)**. Сохраните.
3. Через 1–2 минуты сайт будет на `https://<логин>.github.io/<репозиторий>/`

### Через git
```bash
git add .
git commit -m "update"
git push
```
Репозиторий уже инициализирован; remote — `https://github.com/BelMihMed/calc.git`.
Если remote ещё не добавлен:
```bash
git remote add origin https://github.com/BelMihMed/calc.git
git push -u origin main
```

## Встраивание через iframe (в т.ч. в Битрикс24)
```html
<iframe src="https://<логин>.github.io/<репозиторий>/"
        style="width:100%;border:0;min-height:1600px"
        title="Калькулятор BitrixGPT" loading="lazy"></iframe>
```
`min-height` подберите под высоту контента.

## Пересборка лендингов (необязательно)
Скрипты берут за основу `golos-bitrix24.html` (общий шаблон стиля) и подставляют контент:
```bash
python3 build/build_landings.py         # -> assistant / zapis / documents
python3 build/build_landing_support.py  # -> podderzhka
```
